# 💒SpaceUs
공간 대여 사이트 프로젝트

### 개요
사람과 사람을 연결하고 공간을 공유하는 사이트 구축
- 개발 기간: 2020.09.02 ~ 2020.10.23
- 개발 인원: 4명

### 사용 기술
* Spring
* MyBatis
* JAVA
* JSP
* JavaScript
* SQL
* HTML/CSS
* Git
* AWS

### 담당 기능
* 메인페이지 사용자 검색 입력 창 (AJAX의 autocomplete사용✨)
* 공간 목록 리스트 정렬 순서
* 게시판 제작 (게시판 등록 및 수정, 삭제)
* 게시물 신고 & 블랙리스트 처리
* 댓글과 대댓글 등록 및 신고
* 비밀글 처리
* 관리자 페이지의 회원관리, 공간 승인, 블랙 리스트 관리
